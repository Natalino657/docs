The Orders tab shows every order you have placed while signed in to ShopFront. You can check payment and delivery status at a glance, and tap into any order for a full breakdown.

<Note>
  Orders only appear here after you place them while signed in. Orders placed as a guest, or on a different account, will not appear in this list.
</Note>

## Viewing your order history

Tap the **Orders** tab in the bottom navigation bar. If you are not signed in, you will be prompted to log in first.

Your orders are listed in a table showing the following for each entry:

| Column | Description |
|--------|-------------|
| **#** | The row number for this order in your list |
| **Total** | The total amount charged for the order |
| **Paid** | The date payment was confirmed, or a red indicator if payment has not been received |
| **View** | Tap the eye icon to open the full order detail screen |

## Viewing order details

Tap the eye icon next to any order to open its detail screen. The detail screen is divided into several sections:

- **Shipping details** — the name, email, and shipping address associated with the order
- **Payment method** — the payment method used (for example, PayPal)
- **Payment status** — whether the order has been paid and, if so, the date and time payment was confirmed
- **Delivery status** — whether the order has been delivered and, if so, the date and time it was marked as delivered
- **Items ordered** — a list of every product in the order
- **Order summary** — a price breakdown including items, shipping, tax, and the total

### Order status reference

| Status | Meaning |
|--------|---------|
| **Paid** | Payment was successfully captured |
| **Not Paid** | Payment has not yet been received; you may still be able to complete payment from the order detail screen |
| **Delivered** | The order has been marked as delivered |
| **Not Delivered** | The order is still being prepared or is in transit |

## If an order is missing

If you placed an order but it does not appear in your list, make sure you are signed in to the same account you used when you checked out. If the order still does not appear after refreshing, contact ShopFront support with your order confirmation details.
